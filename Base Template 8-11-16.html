APIs for Dummies

APIs today are more popular than ever before. Not having a public API today is like not having a website in the late 1990s. This Dummies guide will ease you into the world of APIs by teaching you to:

  • Apply the power of APIs to business challenges
  • Define the product nature of modern APIs
  • Manage which APIs to provide or consume
  • Build an effective API technology platform
